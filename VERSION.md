# ASIMOV AI Governance Audit Tool

## Version 0.5 

This version represents a significant milestone in the development of the ASIMOV AI Governance Audit Tool, with particular focus on the Life-Wise Insights system and user experience improvements.

### Key Features
- Fully rebuilt insight generation system
- Progress tracking and improved navigation
- Advanced testing framework
- Comprehensive control-specific templates

### Technical Architecture
The ASIMOV AI Governance Audit Tool consists of:
1. A Flask-based web application
2. SQLite database for audit controls and responses
3. Custom insight generation system
4. Test suites for verification

### Framework Support
The tool currently supports these AI governance frameworks:
- EU AI Act
- NIST AI Risk Management Framework 
- ISO/IEC 42001
- And other major regulatory frameworks
