Database Schema Reference

controls

The main table of AI governance framework controls.

  • id - INTEGER PRIMARY KEY
  • control_name - TEXT
  • category - TEXT
  • framework - TEXT
  • explainability - TEXT
  • description - TEXT
  • evidence - TEXT
  • risk_level - TEXT

responses

User responses from audit questionnaires.

  • id - INTEGER PRIMARY KEY
  • session_id - TEXT
  • control_id - INTEGER
  • framework - TEXT
  • category - TEXT
  • response_score - INTEGER (1-4)
  • reference_text - TEXT
  • timestamp - DATETIME

audit_sessions

Records of individual audit sessions.

  • session_id - TEXT PRIMARY KEY
  • session_name - TEXT
  • framework_filter - TEXT
  • category_filter - TEXT
  • risk_level_filter - TEXT
  • session_date - DATETIME
